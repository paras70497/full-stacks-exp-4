#  Client-Side Routing Experiment (Exp 3)

This project demonstrates client-side routing using **React Router DOM v6**.

---

##  Author
**Name:** Paras  
**UID:** 23BAI70497

---

##  Features
- **React Router Navigation**: Seamless switching between pages without reloading.
- **Multiple Pages**: Home, Products, and About pages.
- **Consistent Layout**: Shared Navbar and Footer across all pages using `Outlet`.
- **Active Link Styling**: Visual feedback in the navigation bar for the current page.

---

##  Tech Stack
- **Framework:** React + Vite
- **Routing:** React Router DOM v6
- **Styling:** Tailwind CSS

---

##  How to Run
1. **Install dependencies:** `npm install`
2. **Start server:** `npm run dev`
3. **Open:** `http://localhost:5173`
